// Format:
//
//	"item"
//	{ 
//		"item_class" 		"value"
//		"item name"		"value"
//
// 		"key"			"value"
//		...
//	}
//
// Notes:
//	"item" is used solely for debugging
//
// REQUIRED FIELDS:
//	item_class must match an entity classname defined in the DLL
//	item_type_name is the name of the base item class 
//	item_name is the descriptive name displayed to the user
//	item_slot is the slot that this item is placed into in the player loadout
//	model_inventory is the model that's used when displaying this item in an item box in the HUD
//	model_player is the model used to display this item on a player model in the HUD
//  used_by_classes contains a subsection with a list of classes that can use this item. Without this section, any class can use the item.
//
// OPTIONAL FIELDS:
//	item_quality restricts the item to only spawning if the random quality matches the items.
//	min_ilevel & max_ilevel are the minimum & maximum item levels the item
//		will ever appear at. Default is 0 & 100 (i.e. always available)
//
// SPREADS:
// NOTE: The way these are calculated is that each component == sin (degrees/2)
//		"0.00873 0.00873 0.00873"			// 1 DEGREE
//		"0.01745 0.01745 0.01745"			// 2 DEGREES
//		"0.02618 0.02618 0.02618"			// 3 DEGREES
//		"0.03490 0.03490 0.03490" 			// 4 DEGREES
//		"0.04362 0.04362 0.04362" 			// 5 DEGREES
//		"0.05234 0.05234 0.05234" 			// 6 DEGREES
//		"0.06105 0.06105 0.06105" 			// 7 DEGREES
//		"0.06976 0.06976 0.06976" 			// 8 DEGREES
//		"0.07846 0.07846 0.07846" 			// 9 DEGREES
//		"0.08716 0.08716 0.08716" 			// 10 DEGREES
//		"0.13053 0.13053 0.13053" 			// 15 DEGREES
//		"0.17365 0.17365 0.17365" 			// 20 DEGREES
//
